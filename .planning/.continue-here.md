<decisions_made>

- **TF-IDF + name-boost + routing_aliases ClassVar over docstring keyword stuffing** — keeps human-facing docstrings honest; aliases are discoverable, drift-resistant, decoupled from the scorer. Generic-name dampener (0.25× for `use`, `function`, `import`, etc.) stops `scalpel_use_function` from cannibalizing every "function" prompt.
- **Per-symbol move applied in-place via `rope.Project.do`** — batched edits don't converge because `MoveGlobal.get_changes` is computed against the current view.
- **Iterate `reversed(members)`** — rope prepends each moved symbol to the target file; reversing yields the user's listed order, critical for forward decls (`Expr = Union[Num, Add, ...]`).
- **Post-pass `_rewrite_package_reexports`** — rope's MoveGlobal misses package-level `from .OLD import S` re-exports; the post-pass handles them with relative-vs-absolute matching to avoid rewriting `from calcpy import X` (the package) when only `calcpy.calcpy` (the module) was the source.
- **Shadow-mode runs `dry_run=False` against the shadow copy** — a true preview asks "what would happen if I committed?" — running real apply against an isolated copy is strictly more informative than the facade's own `dry_run=True` interpretation, and survives facades whose `dry_run` path lies.
- **Item D ships an audit, not 4 new facades** — the 3-user-request demand gate isn't satisfied; speculative facades accumulate maintenance cost without proportional value. Roadmap doc opens 4 slots with admission procedure.

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Stale MCP server cached at uvx archive | Bumping the engine submodule does not update a running MCP server because the process loaded from `~/.cache/uv/archive-v0/<hash>/` and that cache snapshots a specific SHA at install time. New work appears to "ship" but the live dashboard/tool surface keeps the old code. | advisory | Always document the restart-to-pick-up step in milestone commit messages. Optionally pin engine SHA in `.mcp.json` (`uvx --from git+URL@TAG`) so cache invalidation is keyed on tag bumps. |
| Rope MoveGlobal does not rewrite package re-exports | `from .OLD import S` patterns in `__init__.py` are NOT updated by `MoveGlobal.get_changes`, even though direct importers are. Without a post-pass, per-symbol splits leave the package's public API broken. | advisory | The v1.9.1 commit ships `_rewrite_package_reexports` as the post-pass. Future per-symbol move surfaces (e.g. `scalpel_inline_temp` candidate from the v2.0 roadmap) must reuse this helper. |
| Multi-server scans abort on per-server method-not-found | A single LSP that lacks a method (e.g. ruff's missing `textDocument/documentSymbol`) bubbles up `LSPError` and aborts the scan loop. Symbol-search degrades silently to "not found" even when other servers in the pool would have answered. | advisory | The v1.9.2 side-fix wraps each per-server call in `try/except`. Future multi-server iteration sites should follow the same pattern. |
